//Aritmeticos 

+ - * / // -> (división entera) ** -> (potencia)

//Relacionales 

<       -> menor que
<=      -> menor o igual que
>       -> mayor que
>=      -> mayor o igual que
==      -> igualdad 
!=      -> diferencia

Nota: Todas las operaciones relacionales devuelven un 
valor booleano (es decir, True o False)
